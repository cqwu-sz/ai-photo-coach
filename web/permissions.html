先开三个开关 就能出方案

拾光只在出方案那一刻用这些数据，**不会**上传到第三方、不会持续读取。 每一项任选都可以跳过——跳过的功能会自动降级。

📷

摄像头

用来录你眼前的现场（10 秒环视）。必需，否则没法出方案。

已拒绝。在 iOS：设置 → Safari → 摄像头 → 选「询问」或「允许」； Android：长按地址栏锁图标 → 网站设置 → 摄像头 → 允许。

未请求
🧭

方向感应

让 AI 知道你环视到了哪些角度，建议开。拒绝时会用鼠标/触摸模拟，方向结果仅供参考。

已拒绝。在 iOS：设置 → Safari → 动作与方向访问 → 打开； Android：通常无需单独授权，刷新页面再试。

未请求
📍

位置（粗略）

算太阳方位、黄金时刻倒计时和附近的机位推荐。可选，拒绝后方案仍可出，只是没有光线/POI 加成。位置存本地、四舍五入到 11m 网格。

已拒绝。在 iOS：设置 → Safari → 位置 → 选「询问」或「允许」； Android：长按地址栏锁图标 → 网站设置 → 位置 → 允许。

未请求

每项权限只在你启动「环视」时使用一次；不会后台运行。可在浏览器设置里随时撤销。
