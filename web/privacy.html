隐私政策

最近更新：2026-05-11

本政策解释拾光 · AI 取景者（"本应用"）在 iOS / Web 端会收集哪些信息、用途、以及你如何控制它们。如果你只想看一句话总结：环视画面用于本次出片，关键数据脱敏后协助 AI 进步；本机能算的就不上传，能匿名就不实名，可随时一键删除。

1. 我们收集什么

  • 设备 ID（Keychain UUID）：在你首次启动时本机生成，用于把你产生的数据归到你名下。它不是 Apple IDFA，也不会跨 App 共享。
  • 粗略地理位置：仅在你主动触发分析时使用，并在写入数据库前 round 到 ~11 米精度。我们不持续追踪你的位置。
  • 照片 EXIF + 关键帧：分析阶段用于推荐机位与构图。原始视频不上传，仅提取 8-12 张关键帧（每张 ≤ 2 MB）。
  • 邮箱（可选）：仅当你选择 Sign in with Apple 且 Apple 提供时；用于账号恢复，不发营销邮件。
  • 订阅与购买记录：通过 Apple StoreKit 校验后保存订阅状态、产品 id、到期时间。我们不接触你的支付信息。
  • 诊断日志：API 调用错误码 / 响应延迟。已对 GPS、API key、token 做正则脱敏。

2. 我们 不 收集什么

  • IDFA / 跨 App 跟踪标识
  • 通讯录、短信、健康数据
  • 原始视频 / 完整相册
  • 姓名、电话、身份证等强身份信息（除非你明确选择 Sign in with Apple 并授权姓名）

3. 第三方

  • 视觉模型供应商（Google Gemini / OpenAI 兼容协议）：分析时上传关键帧 + 元数据进行推理。如果你启用 BYOK（Bring Your Own Key），你的 API key 仅用于该次请求转发，永不持久化、永不日志。
  • Apple StoreKit / App Store Server Notifications：用于订阅校验与续订事件。
  • 地图与天气（高德、Open-Meteo 等）：根据粗略经纬度查询附近 POI、太阳轨迹和天气。

4. 你的权利

  • 查看你的数据：在 App 内 "账户与订阅" 查看 user id、登录方式、订阅状态。
  • 导出与删除：点 "删除账户与所有数据"（对应后端 DELETE /users/me），我们立即清除反馈、3D 重建任务、订阅记录、用户机位 UGC。
  • 取消订阅：在 iOS "设置 → Apple ID → 订阅" 取消。

5. 数据保留

  • 主动账号：保留至你删除为止。
  • 匿名账号：连续 30 天未使用后自动清理。
  • 订阅记录：按 Apple 合规要求保留至订阅结束 + 6 个月，仅用于退款/续订对账。
  • 诊断日志：滚动保留 30 天。
